# 第一层：影视飓风风格研究层（一次性）

**文档类型**: 方法论规范
**建立时间**: 一次性建立，长期复用
**适用范围**: 所有影视飓风风格的知识类视频项目

---

## 一、层次定位

本层属于**方法论层**，核心任务是研究影视飓风视频风格，产出风格分析报告和分镜脚本分析，为所有项目提供制作规范。

**特点**：
- 一次性建立，长期受益
- 与具体项目无关
- 为所有后续项目提供统一的制作标准

**与项目层的关系**：
```
影视飓风长视频风格调研层（本层）
    ↓ 提供风格分析报告和分镜脚本分析
选题调研分析层（第二层）
    ↓ 提供选题研究内容
内容生成层（第三层）
```

---

## 二、核心模块

### 模块1：影视飓风风格研究

#### 目的
深入研究影视飓风的视频风格和分镜脚本，产出两个核心报告，为所有项目提供工程化标准。

#### 我输入的内容
- 影视飓风视频YouTube链接（6个或更多）

#### 使用的工具
- NotebookLM (https://notebooklm.google.com/)

#### 执行步骤

1. **创建Notebook**
   - 打开NotebookLM
   - 创建新notebook
   - 命名："影视飓风风格调研"

2. **上传资料**
   - NotebookLM会自动分析YouTube视频内容

3. **输入分析提示词**

**提示词1：生成内容形式与结构分析报告**
   ```
   请分析影视飓风的视频风格，从以下维度分析：

   1. 叙事结构：
      - 钩子设计（开场如何抓住注意力）
      - 冲突建立（如何制造矛盾和问题）
      - 反转点设计（打破观众认知的时刻）
      - 结尾升华（如何情感共鸣）

   2. 信息密度：
      - 每分钟信息点数量
      - 数据展示方式
      - 信息点排列节奏

   3. 视觉语言：
      - 镜头切换频率
      - 数据可视化风格
      - 隐喻使用

   4. 节奏控制：
      - 开场/解释/升华的语速差异
      - 信息点密度变化

   5. 情感曲线：
      - 如何在理性分析中加入情感共鸣
      - 情绪起伏设计

   请输出：影视飓风内容形式与结构分析报告
   ```

**提示词2：生成分镜节奏与脚本分析**
   ```
   请深入分析影视飓风的分镜脚本，从以下维度分析：

   1. 镜头节奏分析：
      - 不同类型镜头的时长分布（开场/解释/升华）
      - 镜头切换频率统计
      - 镜头类型使用比例（实拍/素材/信息图/AI生成）

   2. 脚本结构分析：
      - 配音文案的语言风格特点
      - 技术术语的上下文铺垫方式
      - 数据呈现方式（如何读数据、停顿位置）
      - 段落之间的逻辑衔接

   3. 镜头-配音对应关系：
      - 镜头时长与配音时长的比例
      - 不同信息密度下的镜头选择
      - 画面与配音的配合方式

   4. 具体案例拆解：
      - 选取3-5个典型镜头
      - 分析其画面描述、配音内容、时长、素材来源

   请输出：影视飓风分镜节奏与脚本分析报告
   ```

4. **导出两个报告**
   - 保存为：`影视飓风内容形式与结构分析报告.md`
   - 保存为：`影视飓风分镜节奏与脚本分析.md`

#### 输出物
- `影视飓风内容形式与结构分析报告.md`
- `影视飓风分镜节奏与脚本分析.md`

**核心内容应包含**：

**报告1：影视飓风内容形式与结构分析报告**
1. 叙事结构标准
   - 开场必须有钩子（前30秒抛出问题/冲突）
   - 至少2个反转点
   - 结尾必须有升华

2. 信息密度标准
   - 每分钟4-6个信息点
   - 数据必须可视化呈现
   - 避免纯文字画面

3. 视觉语言标准
   - 镜头切换频率
   - 数据可视化风格
   - 隐喻使用方式

4. 节奏控制标准
   - 镜头时长：开场3-5秒、解释5-8秒、升华8-12秒
   - 开场/解释/升华的语速差异

5. 情感曲线标准
   - 开场：紧张、惊讶
   - 中间：理性、专业
   - 结尾：感动、坚定

**报告2：影视飓风分镜节奏与脚本分析**
1. 镜头节奏数据
   - 不同类型镜头的时长分布统计
   - 镜头切换频率具体数值
   - 镜头类型使用比例

2. 脚本写作规范
   - 技术术语的上下文铺垫模板
   - 数据呈现方式（如何读数据、停顿位置标注）
   - 段落衔接的常用句式

3. 镜头-配音对应规范
   - 镜头时长与配音时长的标准比例
   - 不同信息密度下的镜头选择建议

4. 典型案例拆解
   - 3-5个典型镜头的完整分析
   - 可直接参考的镜头描述模板

## 四、质量标准

### 报告质量标准
- [ ] 每个维度都有量化指标（如："4-6点/分钟"）
- [ ] 包含具体案例参考（如："某视频第3分20秒"）
- [ ] 检查清单可执行（5-8项，每项都是是/否判断）
- [ ] 覆盖叙事、视觉、节奏、情感四个层面
- [ ] 避免主观描述（如："很有冲击力"）

### 内容形式与结构分析报告质量标准
- [ ] 包含叙事、信息密度、视觉、节奏、情感5个维度
- [ ] 每个维度有明确的标准和阈值
- [ ] 有具体视频案例作为参考

### 分镜节奏与脚本分析质量标准
- [ ] 包含具体的镜头时长数据统计
- [ ] 有脚本写作的实用模板和句式
- [ ] 包含3-5个典型镜头的完整拆解
- [ ] 镜头-配音对应关系有明确的数值或比例

## 五、维护与更新

### 何时需要更新本层

**风格变更**：
- 项目风格要求改变（如：从影视飓风调整为其他风格）

**问题反馈**：
- 多次出现同类异常（如：连续3个项目质量不达标）
- 现有规范无法覆盖的新情况

**更新流程**：
1. 修订相关模块
2. 标注版本号（如：V1.0 → V1.1）
3. 明确更新内容及生效时间
4. 旧版本归档
